Block Editor Test

(中文說明在下方) This is a dirty JavaScript and visual programming testing application based on Blockly and Jurassic.

This application is aim for the newbie coders to train their programming skills by dragging code blocks together, or just for testing if an algorithm works.

This project is quite incomplete, and it is welcome everyone to help improve this project. Simply fork me and bring back pull requests with great changes.

Although it is incomplete, you still can try it! Feedback are also welcome! Click here to the release section.

The whole open-sourced project is licensed with Apache License v2.0.

這是一個很簡陋...不, 簡便才對...(嗯哼)的一個小程式, 可以用來給新手程式員練習一下編程技巧, 或者用來測試某算法的可行性. 這個程式採用由 Google 開源推出的 Blockly 視覺化編程工具, 再配上可以進行 JavaScript 實時編譯的 Jurassic 引擎, 使編程練習可以更輕鬆快捷.

這程式的完成度還是不太高, 故此功能上可能會比較缺. 在此也歡迎對這程式有興趣的高手們協助改良這個程式, 原始碼是開放的, 可以隨便復刻出來(Fork), 改好後再請求拉取(Pull Request)吧.

儘管這不算完美的完成品, 大家還是可以去這裡下載最新版, 當然如果有意見回饋是最好啦~
